Dijkstra's Algorithm

Introduction

Dijkstra's Algorithm is a fundamental graph algorithm used to find the shortest path between nodes in a weighted graph.

It is widely applied in network routing, geographic mapping, and various optimization problems.

This project aims to implement the algorithm, test it on different datasets, and evaluate its performance.

Objectives

  • Understand the principles behind Dijkstra's Algorithm.
  • Implement the algorithm in a suitable programming language.
  • Evaluate the algorithm's performance on diverse graphs.
  • Analyze the time and space complexity.

Scope of Work

  • Study the theoretical background.
  • Write efficient code in Python/Java/C++.
  • Generate multiple weighted graph test cases.

Methodology

  • Literature review and algorithm study.
  • Software development with VS Code, PyCharm, etc.
  • Testing and debugging on datasets.
  • Use visualization tools to present results.

Timeline

Week Activities
1–2 Literature review & planning
3–4 Coding & initial testing
5 Performance analysis & refinement
6 Report writing & presentation

Expected Outcomes

  • Functional implementation of Dijkstra's Algorithm.
